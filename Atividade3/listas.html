Passo a passo: Configurar um Agente Zabbix em um Servidor Linux

  1. Atualize o sistema com sudo apt update && sudo apt upgrade
  2. Instale o agente: sudo apt install zabbix-agent
  3. Edite o arquivo de configuração: /etc/zabbix/zabbix_agentd.conf
  4. Configure o IP do servidor Zabbix na diretiva Server=
  5. Reinicie o agente: sudo systemctl restart zabbix-agent
  6. Habilite o serviço para iniciar com o sistema: sudo systemctl enable zabbix-agent
  7. No painel do Zabbix, adicione o host e vincule os templates
